# バー検索アプリ 機能要件詳細

## ユーザー種別
- **一般ユーザー**: Bar検索、レビュー投稿、お気に入り登録などを行うユーザー
- **管理者**: Bar情報の登録・編集・削除を行う管理者

---

## 一般ユーザー機能一覧

| 機能 | 詳細 |
|------|------|
| アカウント作成 | メールアドレスで登録 OR Google/X SSOで登録<br>**登録時の入力項目:**<br>- メールアドレス（メール登録の場合は必須）<br>- パスワード（メール登録の場合は必須）<br>- ニックネーム（必須）<br>- 年齢（必須）<br>- アイコン写真（任意） |
| ログイン | メールアドレス + パスワードでログイン OR Google/X SSOでログイン |
| ログアウト | セッションを終了してログアウト |
| パスワードリセット | メールアドレスを入力してパスワードリセット用のメールを受信 |
| マイページでユーザー情報を閲覧 | **表示項目:**<br>- メールアドレス<br>- ニックネーム<br>- 年齢<br>- アイコン写真 |
| マイページでユーザー情報を編集 | **編集可能項目:**<br>- メールアドレス<br>- ニックネーム<br>- 年齢<br>- アイコン写真 |
| Barを検索してBar一覧を表示 | **検索条件:**<br>- 都道府県で絞り込み<br>- 市区町村で絞り込み<br><br>**一覧表示:**<br>- Bar情報をカード形式で表示<br>- 各カードには店名、写真、エリア、評価などを表示 |
| Barの詳細情報を閲覧 | **表示項目:**<br>- 店名<br>- 住所<br>- 店内写真（複数枚）<br>- 営業時間<br>- 定休日<br>- メニュー（ビール・ウィスキー・カクテル）<br>- 各メニューの料金（¥〇〇〇～形式で表示） |
| 投稿されたBarのレビューを閲覧 | Bar詳細画面にレビュー一覧を表示<br>各レビューには投稿者、星評価、コメントを表示 |
| Barをお気に入りに登録 | ハートマークをクリックしてお気に入りに追加<br>※ログインユーザーのみ利用可能 |
| お気に入りのBar一覧を閲覧 | マイページからお気に入りに登録したBar一覧を表示 |
| Barのレビューを投稿 | Bar詳細画面からレビューを投稿<br>**入力項目:**<br>- 星評価（1-5）<br>- コメント<br>※ログインユーザーのみ利用可能 |
| 投稿したBarのレビューを編集 | 自分で投稿したレビューのみ編集可能<br>**編集可能項目:**<br>- 星評価<br>- コメント |
| 投稿したBarのレビューを削除 | 自分で投稿したレビューのみ削除可能 |

---

## 管理者機能一覧

| 機能 | 詳細 |
|------|------|
| Bar情報の登録 | **入力項目:**<br>- 店名（必須）<br>- 説明<br>- 都道府県（必須）<br>- 市区町村（必須）<br>- 住所詳細（必須）<br>- 店内写真（複数枚アップロード可能）<br>- 営業時間<br>- 定休日<br>- ビール料金<br>- ウィスキー料金<br>- カクテル料金<br>- 電話番号<br>- Webサイト |
| Bar情報の編集 | 登録済みのBar情報を編集<br>全ての項目を変更可能 |
| Bar情報の削除 | 登録済みのBarを削除<br>※関連するレビューやお気に入りも連動して削除するか検討 |
| Bar一覧の管理 | 登録されている全てのBarを一覧表示<br>検索・フィルタ機能も利用可能 |

---

## 権限設計

### 一般ユーザー
- 自分のプロフィール情報の閲覧・編集
- 自分が投稿したレビューの編集・削除
- 自分のお気に入り情報の閲覧・追加・削除
- 全てのBar情報の閲覧（読み取り専用）
- 全てのレビューの閲覧（読み取り専用）

### 管理者
- 一般ユーザーの全ての権限
- Bar情報の登録・編集・削除

---

## 制約事項

### ビジネスルール
1. **お気に入り登録**: 同じユーザーが同じBarを重複してお気に入り登録できない（UNIQUE制約）
2. **レビュー投稿**: 1つのBarに対して、1ユーザーが複数のレビューを投稿可能（制限なし）
3. **レビュー編集・削除**: 自分が投稿したレビューのみ編集・削除可能
4. **ログイン必須機能**:
   - レビュー投稿・編集・削除
   - お気に入り登録・削除
   - プロフィール閲覧・編集
5. **管理者権限必須機能**:
   - Bar情報の登録・編集・削除

### データ制約
1. **ニックネーム**: 必須、最大50文字
2. **年齢**: 必須、1-150の整数
3. **星評価**: 必須、1-5の整数
4. **レビューコメント**: 最大2000文字
5. **画像ファイル**:
   - サポート形式: JPEG, PNG, WebP
   - 最大ファイルサイズ: 5MB
   - 最大アップロード枚数: Bar写真は10枚まで
